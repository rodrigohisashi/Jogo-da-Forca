Para compilar todos os arquivos
digitar no cmd g++ *.cpp -o forca.exe
.\forca.exe

Gerar uma unidade de traducao (translate unity) que sao
arquivos .o, para cada um dos arquivos do projeto >>
g++ -c *.cpp

essa unidade de traducao contem as definicoes das funcoes
do arquivo e os dados que aquele arquivo define. 
A partir desses objetos devemos compilar (juntar todos)
e jogar no forca.out.

g++ *.o -o forca.exe (fica quase instantaneo, só junta)

Caso voce queira compilar apenas uma parte do projeto:

g++ -c cabecalho.cpp 

Ai é só juntar novamente
g++ *.o -o forca.exe 
.\forca.exe
